## Getting Started

### Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download)
- Git

### How to Run

```bash
git clone https://github.com/JA-WRI/Tetris-Game.git
cd Tetris-Game/Tetris
dotnet build
dotnet run
```
